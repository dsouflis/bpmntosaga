# bpmntosaga
> BPMN to Saga

v0.0.16 [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Description

Create Redux sagas from BPMN diagrams. Create client objects from Swagger.

## Install

```bash
npm install --save bpmntosaga
```

## Usage

```javascript
import { checkdiagram, diagramToSaga, swaggerToApi } from 'bpmntosaga';

```
